# Tim Walker consulting site

Static GitHub Pages site for Tim Walker's independent consulting work in HDD, nearline storage, hyperscale cloud storage, and HDD-adjacent infrastructure.

## Files

- `index.html` — site content and structure
- `styles.css` — responsive styling
- `.nojekyll` — serves the site as plain static files

## Publishing

Because this repository is named `ttwalkertt.github.io`, GitHub Pages can publish it at:

`https://ttwalkertt.github.io/`

In the repository settings, open **Pages** and select deployment from the `main` branch at the repository root if Pages is not already enabled.

A custom domain can be added later in the same Pages settings after DNS is configured.
